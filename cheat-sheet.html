Ruby Cheat Sheet

If/Else Statements

      def mac_and_cheese(cheese_oz)
        if cheese_oz < 8
          puts "Yeah right, put more cheese in"
        elsif cheese_oz < 12
          puts "It's almost cheesy enough"
        else
          puts "You're on the way to cheese heaven"
        end
      end

Basic conditional statement that excutes a given expression based on input passed to the containing method.

While Loops

      n = 1
      while n < 10
        puts n
        n += 1
      end

While a given condition is true, the block within the statement will continue to loop

REMEMBER!

When comparing multiple pieces of information within a container object, use a placeholder to store a default value. You need something outside of the container to compare the contents against!

Case Statements

      Case cheese
        when "roquefort"
          puts "Salty funky goodness"
        when "camembert"
          puts "Ah yes, refined sweaty-sock essence"
        when "triple-creme"
          puts "Brie who?"
        end

Conditional method when one of several possible arguments is expected. Operators used in if/else statements also apply here.

Until Loops

      n = 1
      until n > 10
        puts n
        n += 1
      end

Until a given condition is met, the code within the loop will execute

.each & .map

      array.each {|i| #do something to each index}

The return value of .each is the receiver. In this case, an array called "array". .each does NOT create a new object. 

      new_array = array.map {|i| #do something to each index}

Unlike .each, .map returns a NEW ARRAY. If you need to store the results, set a new object equal to array.map.
